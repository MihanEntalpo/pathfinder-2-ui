# [PFS.aspx.htm](../PFS/PFS.aspx.md)[Quick Shield Block](./Feats.aspx-ID=4816.md) Feat 8

[Champion](../Trait/Traits.aspx-ID=811.md) [Fighter](../Trait/Traits.aspx-ID=601.md)
Source [Player Core pg. 147](../Source/Sources.aspx-ID=216.md), [Player Core 2 pg. 97](../Source/Sources.aspx-ID=227.md)
Archetypes [Bastion](../Archetypes/Archetypes.aspx-ID=240.md)*, [Viking](../Archetypes/Archetypes.aspx-ID=268.md)*
Prerequisites [Shield Block](./Feats.aspx-ID=5212.md)
* This archetype offers Quick Shield Block at a different level than displayed here.

---

You can bring your shield into place with hardly a thought. At the start of each of your turns, you gain an additional reaction that you can use only to Shield Block.

## Archetype Use

This feat can be used for one or more archetypes in addition to the listed classes. When selected this way, the feat does not have its class traits.
