# [Hefty](./Traits.aspx-ID=478.md)

Source [Treasure Vault pg. 219](../Source/Sources.aspx-ID=191.md)
A hefty shield is so heavy that raising it takes more effort. Raising a Shield with the hefty trait is a two-action activity unless your Strength modifier equals or exceeds the number with the trait.
