# [PFS.aspx.htm](../PFS/PFS.aspx.md)[Advanced Domain](./Feats.aspx-ID=4666.md) Feat 8

[Cleric](../Trait/Traits.aspx-ID=554.md)
Source [Player Core pg. 117](../Source/Sources.aspx-ID=216.md)
Archetypes [Campfire Chronicler](../Archetypes/Archetypes.aspx-ID=300.md), [Soul Warden](../Archetypes/Archetypes.aspx-ID=173.md)*, [Vindicator](../Archetypes/Archetypes.aspx-ID=285.md)
Prerequisites [Domain Initiate](./Feats.aspx-ID=4644.md)
* This archetype offers Advanced Domain at a different level than displayed here.

---

Your prayers have unlocked deeper secrets of your deity’s domain. You gain an advanced domain spell from one of your [domains](../Domain/Domains.aspx.md) for which you have an initial domain spell.

Special You can select this feat multiple times. Each time, you must select a different advanced domain spell.

## Archetype Use

This feat can be used for one or more archetypes in addition to the listed classes. When selected this way, the feat does not have its class traits.
