# [PFS.aspx.htm](../PFS/PFS.aspx.md)[Cognitive Loophole](./Feats.aspx-ID=4981.md) [reaction] Feat 16

[Rogue](../Trait/Traits.aspx-ID=687.md)
Source [Player Core pg. 176](../Source/Sources.aspx-ID=216.md)
Archetype [Shadowdancer](../Archetypes/Archetypes.aspx-ID=77.md)*
Trigger Your turn ends
Requirements You are currently affected by a mental effect
* This archetype offers Cognitive Loophole at a different level than displayed here.

---

You can find a loophole in a mental effect to temporarily overcome it. Until the end of your next turn, you ignore a single mental effect. You can suppress a particular effect using Cognitive Loophole only once. You can’t use Cognitive Loophole to suppress an effect you chose to have affect you.

Special You can use this reaction even if the mental effect is preventing you from using reactions.

## Archetype Use

This feat can be used for one or more archetypes in addition to the listed classes. When selected this way, the feat does not have its class traits.
