# [Tripkee](./Traits.aspx-ID=793.md)

Source [Player Core 2 pg. 319](../Source/Sources.aspx-ID=227.md)
A creature with this trait is a member of the [tripkee](../Ancestry/Ancestries.aspx-ID=84.md) ancestry. Tripkees are a family of froglike humanoids. An ability with this trait can be used or selected only by tripkees. An item with this trait is created and used by tripkees.
