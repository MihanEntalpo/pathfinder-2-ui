## Site Updates

We've made a large change to the site's underlying .NET infrastructure, modernizing many components which hadn't changed in many years. The gains are an easier-to-maintain website and improved performance, which we hope will clear up many of the errors some users see when visiting the site at the early Pacific hours. Some things are unavailable at the moment with the change, see below.

- Due to the change, Shelyn's Corner has a different look but overall the same function.
- The Deck of Many Things feature and the Earn Income calculator will be broken; We plan to prioritize fixing the Earn Income Calculator as that seems to have frequent use. If you use the Deck of Many Things frequenty, please let us know in our feedback channels so we can prioritize it properly.

- A new [Draconic Benefactors](../DraconicBenefactors/DraconicBenefactors.aspx.md) page has been created to help with cataloguing the massive list of dragon sources.

## Site Updates

- For PFS players, we now have scaling encounters built into the Archives. You can find this by clicking "View Encounters" on any PFS adventure on [this page](../Setting/Setting.aspx-Group=40.md). Here's an example for [PFS 7-01](../Encounters/Encounters.aspx-Source=278.md). On this new area you can set which level of encounters you would like to see for your PFS group and we've included a convenient way to print this out.
- Please note that we do not intend to expand this to APs or PFS scenarios prior to Year 7 due to the amount of work it would add. We are doing this specifically for PFS so we can maintain easy access to stat blocks in one place, which we had before year 7. Additionally, Starfinder fans can look forward to this feature coming to our SF2e site once they start using the Alien Core stat blocks.
