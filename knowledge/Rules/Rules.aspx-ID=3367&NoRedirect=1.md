# Chapter 2: Ancestry NPCs

Source NPC Core pg. 167

This chapter presents entries for every ancestry appearing in Pathfinder Player Core and Pathfinder Player Core 2. Each one includes several stat blocks and ideas to spice up your NPCs, come up with quick names, or evoke the ancestries' cultures. If an ancestry also has a section in Pathfinder Monster Core, that information is included in the entry's introductory text.
