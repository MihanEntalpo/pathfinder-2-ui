# [Crossbow](./WeaponGroups.aspx-ID=17.md)

Source [Player Core pg. 283](../Source/Sources.aspx-ID=216.md)
The target takes 1d8 [persistent bleed damage](../Condition/Conditions.aspx-ID=86&Redirected=1.md). You gain an item bonus to this bleed damage equal to the weapon's item bonus to attack rolls.
