# [Bottomless Pit](./Hazards.aspx-ID=467.md)Hazard 9

Magical Mechanical Trap
Source [GM Core pg. 101](../Source/Sources.aspx-ID=218.md)
Complexity Simple
Stealth DC 30 (or 0 if the trapdoor is disabled or broken) or [detect magic](../Spell/Spells.aspx-ID=1485.md)
Description An iron trapdoor covers an infinitely deep 10-footsquare pit.

---

Disable DC 28 [Thievery](../Skill/Skills.aspx-ID=50.md) (trained) to remove the trapdoor
AC 28, Fort +12, Ref +12
Trapdoor Hardness 9, Trapdoor HP 36 (BT 18); Immunities critical hits, object immunities, precision damage
Infinite Pitfall [reaction] Trigger A creature walks onto the trapdoor; Effect The triggering creature falls in and continues to fall, potentially forever. That creature can try to [Grab an Edge](../Action/Actions.aspx-ID=2313.md) to avoid falling. The DC to [Climb](../Action/Actions.aspx-ID=2374.md) the walls or Grab an Edge is 26.

The pit contains many handholds, so the falling creature can try to Grab an Edge again every 6 seconds. If the creature succeeds, it can start to Climb out from that point (though it might be a very long climb, depending on how far the creature fell). Since the creature falls endlessly, it can rest and even prepare spells while falling, though items dropped while falling are usually lost forever.

---

Reset The trap still causes creatures to fall forever if they fall in, but the trapdoor must be reset manually for the trap to become hidden again.
