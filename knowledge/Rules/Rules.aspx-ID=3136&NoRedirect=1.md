# Using Items

Source [GM Core pg. 219](../Source/Sources.aspx-ID=218.md)

This section presents the rules for how characters use alchemical items, magic items, and other special items during play. Some items function automatically, while others need to be activated. While you need only swing a [flaming](../Equipment/Equipment.aspx-ID=2838.md) [greataxe](../Weapon/Weapons.aspx-ID=376.md) to scorch foes, you need to invest a [crown of intellect](../Equipment/Equipment.aspx-ID=3010.md) for it to work, imbibe an [elixir of life](../Equipment/Equipment.aspx-ID=2887.md) to heal yourself, activate [clawed bracers](../Equipment/Equipment.aspx-ID=3068.md) to climb up a wall, and [Cast a Spell](../Action/Actions.aspx-ID=2734.md) using a magical scroll.
