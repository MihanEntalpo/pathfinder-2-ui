# [Catfolk](./Traits.aspx-ID=746.md)

Source [Player Core 2 pg. 315](../Source/Sources.aspx-ID=227.md)
A creature with this trait is a member of the [catfolk](../Ancestry/Ancestries.aspx-ID=77.md) ancestry. Catfolk are humanoids with [feline](../MonsterFamily/MonsterFamilies.aspx-ID=376.md) features. An ability with this trait can be used or selected only by catfolk. An item with this trait is created and used by catfolk.
