# Mythic Feats

Source [War of Immortals pg. 81](../Source/Sources.aspx-ID=232.md)
You can select these feats using the additional mythic feats you gain for being a mythic character. This lasts until 12th level, when you must choose a feat for your mythic destiny. Characters who don't have a mythic Calling can't select mythic feats.
