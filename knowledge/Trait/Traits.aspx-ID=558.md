# [Companion](./Traits.aspx-ID=558.md)

Source [GM Core pg. 272](../Source/Sources.aspx-ID=218.md)
An item with this trait can be worn by an [animal companion](../AnimalCompanions/AnimalCompanions.aspx.md) or similar creature. A companion can have up to two items invested.
