# [Lizardfolk](./Traits.aspx-ID=760.md)

Source [Player Core 2 pg. 317](../Source/Sources.aspx-ID=227.md)
A creature with this trait is a member of the [lizardfolk](../Ancestry/Ancestries.aspx-ID=81.md) ancestry, a family of reptilian humanoids. An ability with this trait can be used or selected only by lizardfolk.
