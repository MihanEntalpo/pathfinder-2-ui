# [Modular](./Traits.aspx-ID=807.md)

Source [Player Core 2 pg. 274](../Source/Sources.aspx-ID=227.md)
The weapon has multiple configurations that you can switch between using an [Interact](../Action/Actions.aspx-ID=2297.md) action. Typically, switching between configurations of a modular weapon allows it to deal different types of damage (listed in the trait, such as “modular B, P, or S”), though it's possible for a modular weapon's description to list more complicated configurations.
