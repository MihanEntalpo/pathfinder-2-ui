# PF2.ru Snapshot

Эта папка содержит локальный снапшот разделов правил Pathfinder 2 Remaster с `https://pf2.ru/rules`.

Содержимое:
- `snapshots/rules/index.html` — оглавление раздела `/rules`.
- `snapshots/rules/**` — скачанные страницы правил.
- `snapshots/ancestries/**` — ancestry-страницы (`?tab=details`) для генерации default-данных.
- `manifest.json` — дата генерации и количество файлов в снапшоте.

Скрипт обновления: `scripts/download_pf2_rules.sh`
