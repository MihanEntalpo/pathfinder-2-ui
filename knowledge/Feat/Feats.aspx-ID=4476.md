# [PFS.aspx.htm](../PFS/PFS.aspx.md)[General Training](./Feats.aspx-ID=4476.md) Feat 1

Human
Source [Player Core pg. 64](../Source/Sources.aspx-ID=216.md)

---

Your adaptability manifests in your mastery of a range of useful abilities. You gain a 1st-level [general feat](./Feats.aspx-Traits=78.md). You must meet the feat’s prerequisites, but if you select this feat during character creation, you can select the feat later in the process in order to determine which prerequisites you meet.

Special You can select this feat multiple times, choosing a different feat each time.

## Traits

Human: A creature with this trait is a member of the human ancestry. Humans are a diverse array of people known for their adaptability. An ability with this trait can be used or selected only by humans.
