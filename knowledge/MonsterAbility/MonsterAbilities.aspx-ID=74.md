# [Scent](./MonsterAbilities.aspx-ID=74.md)

Source [Monster Core pg. 360](../Source/Sources.aspx-ID=221.md)
Scent involves sensing creatures or objects by smell and is usually a vague sense. The range is listed in the ability, and it functions only if the creature or object being detected emits an aroma (for instance, [incorporeal](../Trait/Traits.aspx-ID=632.md) creatures usually do not exude an aroma).

If a creature emits a heavy aroma or is upwind, the GM can double or even triple the range of scent abilities used to detect that creature, and the GM can reduce the range if a creature is downwind.
