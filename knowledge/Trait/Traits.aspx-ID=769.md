# [Ratfolk](./Traits.aspx-ID=769.md)

Source [Player Core 2 pg. 317](../Source/Sources.aspx-ID=227.md)
A creature with this trait is a member of the [ratfolk](../Ancestry/Ancestries.aspx-ID=82.md) ancestry. Ratfolk are humanoids who resemble rats. An ability with this trait can be used or selected only by ratfolk.
