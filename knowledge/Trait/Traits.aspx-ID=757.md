# [Hobgoblin](./Traits.aspx-ID=757.md)

Source [Player Core 2 pg. 316](../Source/Sources.aspx-ID=227.md)
A creature with this trait is a tall, militaristic goblinoid. [Hobgoblins](../Ancestry/Ancestries.aspx-ID=78.md) tend to have [darkvision](../Rules/Rules.aspx-ID=2410.md). An ability with this trait can be used or selected only by hobgoblins. An item with this trait is created and used by hobgoblins.
