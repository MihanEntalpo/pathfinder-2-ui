# [Tengu](./Traits.aspx-ID=776.md)

Source [Player Core 2 pg. 318](../Source/Sources.aspx-ID=227.md)
A creature with this trait is a member of the [tengu](../Ancestry/Ancestries.aspx-ID=83.md) ancestry. Tengu are humanoids who resemble crows. An ability with this trait can be used or selected only by tengu. An item with this trait is created and used by tengu.
