# Potions

Source [GM Core pg. 259](../Source/Sources.aspx-ID=218.md)

A potion is a magical liquid that is activated when you drink it, which uses it up. Potions have the [potion](../Trait/Traits.aspx-ID=672.md) trait. You can activate a potion with an [Interact](../Action/Actions.aspx-ID=2297.md) action as you drink it or feed it to another creature. You can feed a potion only to a creature that is within reach and willing or otherwise so helpless that it can’t resist. You usually need only one hand to consume a potion or feed it to another creature.
