# [Low-Light Vision](./MonsterAbilities.aspx-ID=69.md)

Source [Monster Core pg. 359](../Source/Sources.aspx-ID=221.md)
The monster can see in dim light as though it were bright light, so it ignores the [concealed](../Condition/Conditions.aspx-ID=62.md) condition due to dim light.
