# [PFS.aspx.htm](../PFS/PFS.aspx.md)Living MantleItem 10+

[Focused](../Trait/Traits.aspx-ID=609.md) [Invested](../Trait/Traits.aspx-ID=637.md) [Plant](../Trait/Traits.aspx-ID=668.md) [Primal](../Trait/Traits.aspx-ID=676.md)
Source [GM Core pg. 292](../Source/Sources.aspx-ID=218.md)
Usage worn cloak; Bulk L

---

The base of this cloak is a thick layer of moss, but it slowly picks up native plants from each area it spends time in. You gain a +2 item bonus to Nature checks. You also suffer no effects from severe cold and severe heat.

Activate—Druidic Secrets [free-action] ([concentrate](../Trait/Traits.aspx-ID=561&Redirected=1.md)); Frequency once per day; Effect You gain 1 Focus Point, which you can spend only to cast an order spell. If you don't spend this Focus Point by the end of this turn, it is lost.

Craft Requirements You are a druid.
