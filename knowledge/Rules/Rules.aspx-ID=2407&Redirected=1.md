# Imprecise Senses

Source [Player Core pg. 433](../Source/Sources.aspx-ID=216.md)

Average hearing is an imprecise sense—it can’t detect the full range of detail that a precise sense can. You can usually sense a creature automatically with an imprecise sense, but it has the [hidden](../Condition/Conditions.aspx-ID=79.md) condition instead of the [observed](../Condition/Conditions.aspx-ID=84.md) condition. It might be undetected by you if it’s using [Stealth](../Skill/Skills.aspx-ID=48.md) or is in an environment that distorts the sense, such as a noisy room in the case of hearing. In those cases, you have to use the [Seek](../Action/Actions.aspx-ID=2301.md) basic action to detect the creature. At best, an imprecise sense can be used to make an undetected creature (or one you didn’t even know was there) merely hidden—it can’t make the creature observed.
